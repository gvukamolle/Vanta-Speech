# Search operators (Google/Bing style)

## Core operators
- Exact phrase: "exact phrase"
- Exclude terms: -word
- OR alternatives: term1 OR term2
- Site filter: site:example.com
- File type: filetype:pdf (or doc, ppt, xls)
- In URL: inurl:api
- In title: intitle:"release notes"

## Power combos
- Official docs only: site:developer.android.com "Compose" "migration"
- Tech doc + PDF: "Confluence REST" filetype:pdf site:atlassian.com
- Release notes: "product name" "release notes" 2025..2026
- GitHub issues: site:github.com "project" "issue" "bug"

## Recency hints
- Use a year range in the query: 2025..2026
- Add month name for narrow windows: "January 2026"

## Language variants
- English first for accuracy.
- Add local language terms if needed for regional sources.
