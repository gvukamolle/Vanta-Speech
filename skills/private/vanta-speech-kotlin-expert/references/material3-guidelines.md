- Navigation: https://m3.material.io/foundations/navigation/overview

## Quick reminders
- Favor system type scales and responsive typography.
- Keep layouts spacious and use consistent padding.
- Support Dynamic Color and Dark Mode where possible.
- Keep motion subtle and tied to state changes.
- Use native navigation patterns for phones/tablets/foldables.
